# PDS Viewer

This is a frontend for the Planetary Data System of NASA JPL.

![Screenshot of the application showing an image made by Cassini.](./screenshot.png)

# Usage

## Desktop App

1. Clone the repo
2. `yarn install`
3. `yarn build`
4. Open the file in the `/out`

## Frontend Only

1. Clone the repo
2. `yarn install`
3. `yarn react:build`
4. The files will be in `/dist`
